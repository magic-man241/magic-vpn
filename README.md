## 🔥 Qu'est-ce que MAGIC-VPN ?

MAGIC-VPN est un script Termux ultra-puissant conçu pour **analyser, modifier et optimiser** tes configurations VPN (V2Ray, VLESS, VMess, Trojan, Shadowsocks, HTTPCustom, NpvTunnel...).  
Il te permet de changer un **host** ou un **SNI** en une seconde, de stabiliser ta connexion, et bien plus encore.

---

## ✨ Fonctionnalités

- 🛡️ **Analyse intelligente** : Détecte automatiquement le type de configuration (VLESS, VMess, Trojan, SS, HTTPCustom, NpvTunnel...)
- 📋 **Liste des applis compatibles** : Pour chaque type, le script t'indique les applications Android qui fonctionneront (v2rayNG, NapsternetV, HTTPCustom, DarkTunnel, NpvTunnel, NetMod...)
- ✏️ **Modification Host/SNI** : Change l'adresse ou le SNI d'un lien en une seule commande. Le résultat est affiché proprement, prêt à être copié.
- 🚀 **Stabilisation VPN** : Optimisations réseau sans root (DNS Cloudflare, keepalive ping)
- 💾 **Sauvegarde/Restauration** : Stocke tes configurations importantes en toute sécurité
- 📜 **Logs détaillés** : Suis toutes les actions effectuées
- 🔄 **Mise à jour automatique** : L'option 7 vérifie si une nouvelle version est disponible sur ce dépôt GitHub, et met à jour le script en un clic
- 🔐 **Protection par mot de passe** (mot de passe : `idiot`)
- 🎨 **Interface aux couleurs du Gabon** (vert, jaune, bleu) avec logo centré

## 📲 Installation

1. Installe [Termux](https://termux.dev/) (de préférence via F-Droid)
2. Dans Termux, copie et colle cette commande :

   pkg update -y && pkg upgrade -y && pkg install -y curl && bash <(curl -sL https://raw.githubusercontent.com/magic-man241/magic-vpn/main/magic.sh)

3. Le script s'installe automatiquement. Ensuite, tape simplement :

   magic

4. Mot de passe : `idiot`
